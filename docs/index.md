# NeuroCore Documentation

Welcome to the **NeuroCore** documentation.

NeuroCore is a GenAI cluster monitoring and benchmarking platform designed to help users understand system resource utilization and machine learning performance on shared compute infrastructure. 

This documentation provides installation instructions and detailed explanations of the dashboard features.


---

## Getting Started

If you are new to NeuroCore, we recommend the following path:

1. **Install NeuroCore**  
   → Follow the [Installation Guide](installation.md)

2. **Run the Platform**  
   → Use the [Quick Start](quickstart.md) to launch the dashboard and execute a benchmark

3. **Explore the Dashboard**  
   → Learn about resources, jobs, and benchmarking in the [Dashboard Documentation](dashboard/index.md)
